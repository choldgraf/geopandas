geopandas.read_file¶

geopandas.read_file(filename, **kwargs)¶

Returns a GeoDataFrame from a file.

filename is either the absolute or relative path to the file to be opened and kwargs are keyword args to be passed to the open method in the fiona library when opening the file. For more information on possible keywords, type: import fiona; help(fiona.open)
